# 什么是 Loop？什么是平凯 Loop？

最近，黄仁勋提出一个新观点：
Prompt 正在过时，Loop 才是新的 AI 范式。

Prompt （提示词）时代，人教 AI 做事。
Agent （智能体）时代，AI 开始帮人做事。
Loop 时代，人和 AI 一起把事情做成。

Loop不是一次对话
是一个持续迭代的工作闭环
 ⭕️ Prompt：一问一答，完成碎片化任务。
 ⭕️ Loop：人定目标，AI 自主规划、执行、反馈、优化。

它的核心是 “搞定一件事”。
可以说：
Agent 是能力，决定了单兵能做什么。
Loop 是协作，决定了整个团队能完成什么。

平凯 Loop是一个企业级的 Loop产品
人和智能体可以在这里协同工作。

在平凯 Loop 中，你只需要提出目标
多个智能体会围绕目标自主协作：
 ⭕️ 有人负责规划任务
 ⭕️ 有人负责执行
 ⭕️ 有人负责检查结果
 ⭕️ 有人负责持续优化
整个过程形成一个不断运行、不断迭代的 Loop
最终完成你的目标。

平凯 Loop 这个产品本身
也是通过 Loop 的方式实现的。
