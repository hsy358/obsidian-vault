# 港大开源「Agent 版飞书」，人和AI一起上班

港大数据智能实验室（HKUDS）在 6 月 22 日开源了 AgentSpace，一个「人与 AI 员工一起办公」的协作工作区。

它的定位很直白——飞书是给人用的，AgentSpace 是给人和 Agent 一起用的。

Agent 越来越强，但团队用它的方式没跟上。今天大多数 Agent 产品都是为个人设计的，一个人、一个终端、一段对话。

一旦团队想把 Agent 用进日常运营，就处处卡壳——强大的 Agent 藏在某个人的终端里别人看不见，上下文四处散落，换个运行时就要从头重建，凭证和对外动作几乎无法统一审查。

AgentSpace 的做法，是把 Agent 从「工具」变成「数字员工」：人负责方向与授权，Agent 负责协调与执行。它给出四个核心能力——

① 调度：通过 AgentRouter，同一个 Agent 自动挑最合适的运行时（Claude Code、Codex、Hermes、OpenClaw……），身份、技能、权限全程不变，只换执行的那层壳。

② 能力共享：把锁在个人账号里的好 Agent，变成全组织可见、可借用、可转移的数字员工，但负责人始终保留审批权。

③ 多 Agent 协作：取证、预算核对、审批、执行、交付，跨频道自动推进，关键动作走人工审批。

④ 治理：每个动作都有边界、记录和归属，一个控制平面集中管理授权、审批、撤销与审计。

它不是更聪明的聊天机器人，而是一个让人与 Agent 一起把活干完、且全程可控可追溯的运营界面。
