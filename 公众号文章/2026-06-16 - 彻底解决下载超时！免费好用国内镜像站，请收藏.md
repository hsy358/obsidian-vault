# 彻底解决下载超时！免费好用国内镜像站，请收藏
在日常工作中，最让人头疼的莫过于“网络慢”和“连接超时”。这是因为大部分资源都存放在海外站点，受国内网络环境限制很难直接访问。想要解决这类问题，最简单高效的办法就是配置国内镜像站。

这里给大家分享 6 个长期稳定的免费国内镜像站，不用再四处查找，建议收藏备用！

**1. 阿里云开源镜像站 ✅**

- **地址**：https://mirrors.aliyun.com
- **包含资源**：Linux系统镜像、Linux系统软件包源、Docker/K8s安装源、开发语言库（如Maven、NPM、PyPI、Go等）、运维软件安装包。
- **核心优势**：网络稳定、同步频率高、资源全。
**2. 华为开源镜像站 ✅**

- **地址**：https://mirrors.huaweicloud.com
- **包含资源**：Linux系统镜像、Linux系统软件包源、Docker/K8s安装源、开发语言库（如Maven、NPM、PyPI、Go等）、运维软件安装包。
- **核心优势**：ARM和国产化资源全。
**3. 网易开源镜像站**

- **地址**：http://mirrors.163.com
- **包含资源**：Linux系统镜像、K8s安装源

最后给大家分享几个当前可用的 Docker 镜像加速器。Docker 默认直连 DockerHub，国内网络经常下载卡顿、连接超时，配置镜像加速就能大幅提升拉取速度。

编辑配置文件 `/etc/docker/daemon.json`，填入以下加速地址：

-
-
-

```
{  "registry-mirrors": ["https://docker.1ms.run","https://hub1.nat.tf","https://docker.m.daocloud.io"]}
```

重启生效：

-

```
systemctl restart docker
```
