## 🔑 关键发现

### 关于密码
- 5 个项目**都没用默认密码**（这是我之前的部署原则）
- 密码都是何大人你**setup 时自己设的**
- DB 里都是 bcrypt 加密，**我无法查**
- 建议用 1Password / Bitwarden 等密码管理器

### 关于 admin 权限
- Dify：你的账号是 admin
- Langfuse：你的账号**不是 admin**（默认首次注册 ≠ admin）→ 如需要 SQL 提升
- RAGFlow：内置 admin 是 `admin@ragflow.io`，你注册的是普通用户

### 关于服务状态
- 3 个核心服务（Dify / Langfuse / RAGFlow）✅ 运行中
