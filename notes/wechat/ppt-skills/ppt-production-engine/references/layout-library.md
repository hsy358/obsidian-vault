# Layout Library

Map slide types to stable layout patterns.

## Core Layouts

- `cover`: full title zone, subtitle, metadata, optional hero image.
- `executive_summary`: 3-5 conclusion cards.
- `section_divider`: large section title, small transition message.
- `two_column`: left argument, right visual.
- `matrix`: 2x2, 3x3, or comparison table.
- `process_flow`: horizontal or vertical process with 4-8 steps.
- `architecture`: layered boxes and arrows.
- `capability_map`: grouped cards with icons.
- `roadmap`: time axis with phases and milestones.
- `chart_focus`: chart plus insight callout.
- `appendix`: dense table or source evidence.

## Consulting Defaults

- Use conclusion titles.
- Keep a fixed title zone across slides.
- Use one dominant message per slide.
- Put evidence below the message, not before it.
- Prefer cards, matrices, timelines, and diagrams over bullet walls.

## When To Delegate

If one slide becomes a dense system architecture, platform map, capability map, or end-to-end flow, delegate that slide to `editable-architecture-ppt`.
