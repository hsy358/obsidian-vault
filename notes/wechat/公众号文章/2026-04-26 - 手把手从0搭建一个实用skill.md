# 手把手从0搭建一个实用skill
# 我花了一个下午写了个 "邮箱→报销单" 自动化 skill

从此告别月底手敲 Excel

## ▶起点：一个非常具体的痛

每个月底我都要花两小时干同一件事：

- 打开 QQ 邮箱搜「发票」
- 一封封下 PDF/OFD 附件
- 12306、滴滴、京东这种发链接的还得点开页面再下
- 在 Excel 里敲发票号、日期、金额、分类
- 算分类小计 + 总额
- 把个人抬头的剔掉、重复的合并、ZIP 解压
- 提交报销系统
每张都要肉眼对一遍。烦死了。

我跟 Claude Code 说："帮我把这事儿自动化掉"。

## ▶规划：先把方案聊清楚，再写代码

第一轮我没让它写代码，而是先聊架构。最后定了拆两个独立 skill：

- **qq-invoice-fetch** —— 从 QQ 邮箱按日期区间拉发票到 `报销/<月>/`
- **invoice-organize** —— 扫描目录里的 PDF，整理成 Excel + Markdown 清单
理由：单一职责。哪天换 163 邮箱只改前一个，整理逻辑不用动。

接着拍板了几件事：

决策

选择

邮箱授权码存哪

`.env` + gitignore（不进 Keychain，简单）

跑完是否自动接 organize

不自动，先让用户看落盘对不对

链接型邮件怎么办

自动跟链接，跟不到就列 URL

落盘目录

跟 `--since` 的月份走

这一步很关键——很多人写 AI 项目失败是因为上来就写代码，写到一半发现"诶我应该这么设计"。先把决策拍板能省 **80% 返工**。

## ▶踩坑 1：QQ IMAP 的玄学 bug

第一次跑，4 月明明就 100 多封邮件，SEARCH 返回了 9740 封（其实是 INBOX 全部历史邮件）。

写脚本一条条试搜索条件，试出来一个无文档的 bug：

```
# QQ IMAPM.search(None, '(SINCE "01-Apr-2026" BEFORE "26-Apr-2026")')# → 9740 ❌  (BEFORE 被吞)M.search(None, 'SINCE "01-Apr-2026"', 'BEFORE "26-Apr-2026"')# → 116 ✅  (正确)
```

外层括号 `()` 会让 QQ 把第二个条件吞掉。删掉括号瞬间正常。

## ▶踩坑 2：PDF / OFD 互相把对方干掉

中国电子发票一张发票同时有 PDF 版（给人看）和 OFD 版（国标镜像），同发票号。

第一版去重用「发票号」当 key —— 跑完一看： 5 张 PDF 全没了，只剩 OFD（OFD 还读不了内容）。

修复：去重 key 改成 **(发票号, 扩展名)**。PDF 和 OFD 同号都保留。

教训：去重时要想清楚"什么算重复"，不是越严越好。

## ▶链接型邮件的处理

12306 / 滴滴 / 京东 都不直接发 PDF，发链接：

```
解析 HTML → 找 <a> / <iframe>  → 过滤 href 含 pdf / fapiao / download / 发票    → 带浏览器 UA 抓      → Content-Type 是 application/pdf 或开头 %PDF- 才存        → 还是 HTML？再跟 1 跳          → 跟不到就列 URL 让用户点
```

京东特殊处理：它发的是 `.xml` 链接（电子发票全字段 XML，不是 PDF），识别后单独提示「去京东 App 重发 PDF 邮件」。

## ▶整理 → Excel

`invoice-organize` 这边：用 Claude Code 的 Read 工具直接读 PDF（PDF 渲染成图 + 文字一起喂给模型），逐张抽：

- **发票号**（去重主键）
- **开票日期 / 消费日期**（铁路票常常不同）
- **销售方**（铁路票格式："中国铁路 + 车次 + 起止站"）
- **项目**（按 SKILL.md 规则归类：交通/餐饮/住宿/办公用品/其他）
- **价税合计**（取小写值，不用"合计"——后者是不含税）
抽完丢给一个 `make_xlsx.py` 脚本（openpyxl 模板），固定 JSON 输入：

```
{  "title": "4 月报销发票清单",  "rows": [{"序号": 1, "开票日期": "...", ...}],  "categories": [{"类别": "交通", "金额": 308.40}, ...],  "notes": ["..."]}
```

输出蓝色表头 / 金额 ¥1,234.56 / 合计行红色加粗的 Excel。直接提交财务系统。

## ▶强制规则：剔除个人抬头

公司不能报销个人抬头发票（购买方栏写人名而非公司）。skill 自动识别 + 剔除，备注里逐张列出让用户知情。

我 4 月那张银泰百货 650.90 的运动羽绒服就这么被剔了——个人发票，公司报不了。
