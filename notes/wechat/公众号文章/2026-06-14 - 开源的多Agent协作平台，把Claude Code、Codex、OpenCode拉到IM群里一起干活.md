# 开源的多 Agent 协作平台，把 Claude Code、Codex、OpenCode 拉到 IM 群里一起干活
开源的多 Agent 协作平台，把 Claude Code、Codex、OpenCode 拉到 IM 群里一起干活。一个编排者自动拆任务、并行派活给群里的 Agent，干完再验再合并。消息里有 12 种以上内联渲染，diff、网页预览、表格、提交记录，直接在聊天里看还能编辑。
Polynoia，一个 IM 风格的多 Agent 协作平台，让 Claude Code、Codex、OpenCode 等 AI 编码 Agent 在共享工作区中并行协作开发软件。
