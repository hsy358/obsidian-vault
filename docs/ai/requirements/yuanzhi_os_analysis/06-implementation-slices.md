### SLICE-01：资源上传与 PDF 解析基础能力

**优先级**：P0
**预估工时**：5天
**前置依赖**：无

**涉及文件**：

| 区域 | 文件 | 变更类型 | 说明 |
|---|---|---|---|
| 前端 | `GlobalDock.vue` | 修改 | 扩展上传入口，增加解析状态展示 |
| 前端 | 资源上传页 [new] | 新增 | 独立的资源上传页面 |
| 前端 | 资源详情页 [new] | 新增 | 展示解析结果 |
| 后端 | `ResourceController.java` | 修改 | 新增上传接口、解析触发接口、解析状态查询接口 |
| 后端 | `ResourceExtractService.java` | 修改 | 扩展 PDF 解析逻辑，增加 OCR 调用 |
| 后端 | 资源相关 DTO/Entity [new] | 新增 | Resource, ResourceExtracted 实体 |
| 数据库 | `resource` 表 | 新增 | 资源元数据 |
| 数据库 | `resource_extracted` 表 | 新增 | 解析结果指标 |

**交付物**：
- 资源上传功能（支持 PDF/DOC/PPT/图片，≤ 100MB）
- PDF 文本型解析能力（提取课程名、学时、学分、考核方式）
- 图片型 PDF / 图片自动触发 OCR 识别
- 解析结果展示与手动修正功能

**验收标准**：AC-001, AC-002, AC-003, AC-004, AC-020
