## 🎯 PARA 7 大类文件分布

```datacore
TABLE length(rows) AS "文件数"
FROM ""
WHERE file.path != "_excalidraw" AND file.path != "_meta"

```

## 🔥 最近 30 天修改的文件

```datacore
TABLE
```

## 🎮 Datacore 查询小抄

```datacore
TABLE / LIST / TASK
FROM "<路径>"
WHERE <条件>
SORT <字段> ASC/DESC
LIMIT <数量>
```

### 常用函数
- `file.link` — 文件链接
- `file.path` / `file.folder` — 路径
- `file.name` / `file.size` / `file.mtime`
- `file.tags` / `file.inlinks` / `file.outlinks`
- `dateformat(date, "yyyy-MM-dd")` — 格式化日期
- `date(today)` — 今天
- `dur(N days)` — 时长
- `split(str, "delim")[idx]` — 分割取段
- `length(rows)` / `length(arr)` — 数组长度
- `contains(str, "sub")` / `contains(arr, "val")`

### 示例查询
```datacore
# 找 2026-06 修改的所有德勤笔记
LIST
FROM "1-Projects/德勤"
WHERE file.mtime >= date("2026-06-01") AND file.mtime < date("2026-07-01")

# 找所有含 RAG 关键词的笔记
LIST
FROM ""
WHERE contains(file.content, "RAG") OR contains(file.content, "检索增强")

# 找大于 100KB 的大文件
TABLE file.size, file.link
FROM "3-Resources"
WHERE file.size > 100000
SORT file.size DESC
```
