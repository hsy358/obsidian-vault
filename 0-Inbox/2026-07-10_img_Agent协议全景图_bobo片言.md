  5 个协议核心定义：
  - ACP（Agent Context Protocol）/ 应用上下文交互协议 —— 用户/IDE ↔ Agent 的标准通信桥梁（请求、上下文、流式响应、事件通知）
  - MCP（Model Context Protocol）/ 模型上下文协议 —— Agent ↔ 工具/数据库/API 的标准调用方式（参数、结果、错误处理）
  - A2A（Agent-to-Agent Protocol）/ Agent 对 Agent 协议 —— Agent 之间发现、通信、委派任务、共享上下文的标准
  - ARD（Agent Resource Discovery）/ 资源发现协议 —— Agent 自动发现企业内可用资源（工具/API/DB/Agent）的能力描述与使用方式
  - ARA（Agent Resource Registry）/ 资源注册与管理协议 —— 资源的注册、元数据、权限、版本、状态管理 = 企业级治理能力
